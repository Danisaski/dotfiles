# dotfiles
Dotfiles for my development environment

# Clone the dotfiles
```bash
git clone https://github.com/Danisaski/dotfiles.git ~/dev
```

```bash
git clone git@github.com:Danisaski/dotfiles.git ~/dev
```

# Make scripts executable
```bash
cd ~/dev
chmod +x setup.sh scripts/*.sh
```

# Run the setup
```bash
./setup.sh
```
